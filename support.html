Initial Setup

1
When prompted for Location Permission, select "While Using the App" to enable WiFi name display.
2
When prompted for Local Network Permission, tap "Allow" to enable network diagnostics.
3
Connect to a WiFi network and start using all features.
ğŸ’¡ Tip: Location permission is only used to get WiFi network name (required by iOS). We do not record or upload your location.

Using Network Diagnosis

Tap "Start Diagnosis" on the home page. The app will automatically check:

  • WiFi Connection: Whether connected to WiFi
  • Router Connectivity: Gateway connection status
  • DNS Resolution: Domain name resolution
  • Internet Access: External network access

Using Speed Test

1
Go to "Speed Test" feature
2
Tap "Start Test"
3
Wait for download and upload tests (~30 seconds)
4
View results including download speed, upload speed, and latency
